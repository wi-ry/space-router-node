Space Router Home Node

This installer will set up the Space Router Home Node daemon on your Mac.

What gets installed

Binary: /opt/spacerouter/space-router-node
Service: /Library/LaunchDaemons/com.spacerouter.homenode.plist
Config: /etc/spacerouter/spacerouter.env

After installation

The service will start automatically. Edit /etc/spacerouter/spacerouter.env to configure your node, then restart the service.
